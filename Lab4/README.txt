Lab4: RARS Stock Analysis

------------ 
DESCRIPTION 
This program takes in a csv spreadsheet of stock data (names and incomes)
then prints out the total file size in bytes, the total income in stocks,
and the stock with the maximum income.

------------ 

INSTRUCTIONS
This program is intended to be run using the RISCV Assembler and Runtime Simulator(RARS).
